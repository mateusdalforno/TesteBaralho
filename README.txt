TesteBaralho
============

Aqui ficam os testes que são realizados utilizando a biblioteca do baralho.


CONSTRUÇÃO DA BIBLIOTECA

Foi usado para a construção da biblioteca o construtor Apache Ant
<http://ant.apache.org/>. É necessário que ele esteja instalado para que seja
construído.
É necessário que o jar da biblioteca do baralho esteja na pasta lib.
Para construir o executável, acesse o terminal/command, vá até a pasta que possui
o arquivo build.xml, e use o comando "ant". Após concluída a construção, será criada
uma pasta build, que vai conter o .jar da do executável.


EXECUÇÃO DOS TESTES

Acesse a pasta build e coloque o comando java -jar executavel.jar.
Aparecerá na tela todos os testes realizados na biblioteca.
